# Fecha
Se define el tipo de dato fecha

Conjunto de valores
(dia, mes, año, DiaDelAño, NombreDelMes)

dia, mes, DiaDelAño ∈ N.
año ∈ enteros.
NombreDelMes ∈ String.

Conjunto de operaciones 
Sumar
Restar
Dividir ( para calcular un "promedio" entre dos fechas)
Cantidad de dias hasta el dia de la fecha.
